# Fincas-Gonzalo-2

Este proyecto es una aplicación de administración agraria diseñada para facilitar el seguimiento y gestión de la producción agrícola, el tratamiento del cultivo, el riego y el registro de lluvias. Utilizando tecnologías modernas como el MERN Stack (MongoDB, Express, React, Node.js), esta aplicación proporciona una plataforma robusta y fácil de usar para agricultores y profesionales del sector agrícola.

## Características Principales

- **Gestión de Producción**: Registra y realiza un seguimiento detallado de la producción agrícola, incluyendo la siembra, cosecha, rendimiento y calidad de los cultivos.
  
- **Tratamiento del Cultivo**: Permite el registro y seguimiento de los diferentes tratamientos aplicados a los cultivos, como fertilizantes, pesticidas y otros productos fitosanitarios.

- **Sistema de Riego**: Facilita la planificación y monitorización del riego, ayudando a optimizar el uso del agua y asegurar las condiciones óptimas para el crecimiento de los cultivos.

- **Registro de Lluvias**: Registra los datos de precipitación para realizar análisis históricos y planificar las actividades agrícolas en función del clima.


## Tecnologías Utilizadas

- **MongoDB**: Base de datos NoSQL para almacenar los datos de forma eficiente y escalable.
  
- **Express**: Framework de Node.js para construir el backend de la aplicación, proporcionando una estructura sólida para manejar solicitudes HTTP y definir rutas.

- **React**: Biblioteca de JavaScript para construir interfaces de usuario interactivas y dinámicas, permitiendo una experiencia de usuario fluida y atractiva.

- **Node.js**: Entorno de ejecución de JavaScript del lado del servidor que permite construir aplicaciones web escalables y de alto rendimiento.

## Instalación y Uso

1. Clona este repositorio en tu máquina local.
2. Instala las dependencias utilizando el comando `npm install` en el directorio raíz.
3. Configura tu base de datos MongoDB local o remota y asegúrate de que esté en funcionamiento.
4. Crea el archivo `.env` y proporciona los valores de configuración necesarios (como la URL de la base de datos [http://localhost:27107 para local]).
5. Inicia la aplicación utilizando el comando `npm start` en el directorio /client y en otra terminal ejecuta tambien `npm start` para el directorio /server.
6. La aplicación estará disponible en tu navegador en `http://localhost:5173` ya que esta construida con Vite.
